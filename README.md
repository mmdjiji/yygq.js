## 安装

### 局部安装
```shell
$ npm install --save yygq
```

### 全局安装
```shell
$ npm install -g yygq
```

## 导入

### Node.js
```js
import yygq from 'yygq';
```

### HTML
```html
<script src="https://std.ac/yygq.js/yygq.min.js"></script>
```
